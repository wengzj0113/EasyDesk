## 产品简介

EasyDesk 是一款极简远程桌面应用，让你可以轻松实现远程办公、远程协助等功能。

**主要功能：**
- 免登录快速连接 - 输入设备码和密码即可远程控制
- 设备绑定管理 - 登录后可绑定自己的设备，一键直连
- VIP专属服务 - 享受超清画质、不限速传输
- 连接历史记录 - 查看历史连接设备

## 快速开始

### 访问应用

打开浏览器访问：`http://localhost:3003`

### 首页功能

首页展示三个主要功能入口：

| 功能入口 | 说明 |
|---------|------|
| 快速连接 | 免登录，输入设备码+密码即可远程连接 |
| 我的设备 | 登录后绑定自己的设备，一键直连 |
| VIP服务 | 专属线路、超清画质、不限速传输 |

## 账户管理

### 注册账号

1. 点击右上角「注册」按钮
2. 填写注册信息：
   - 用户名：至少3个字符
   - 邮箱：有效的邮箱地址
   - 密码：至少6个字符
3. 点击「注册」完成注册

### 登录账号

1. 点击右上角「登录」按钮
2. 输入用户名和密码
3. 点击「登录」

登录成功后，页面右上角会显示用户名和VIP标识（如已开通VIP）。

### 退出登录

1. 点击右上角用户名
2. 在下拉菜单中选择「退出登录」

## 设备管理

### 设备绑定

登录后，可将远程设备绑定到账户，实现一键直连。

**绑定步骤：**

1. 点击「我的设备」
2. 点击「绑定设备」按钮
3. 在远程设备上运行 EasyDesk，获取设备码
4. 输入设备码和设备名称
5. 点击「绑定」

### 查看已绑定设备

在「我的设备」页面可以查看所有已绑定的设备：

- 设备名称
- 设备码
- 在线状态
- 最后在线时间

### 解除设备绑定

1. 在设备列表中找到要解绑的设备
2. 点击「解绑」按钮
3. 确认解绑操作

### 修改设备密码

1. 在设备列表中找到要修改密码的设备
2. 点击「修改密码」按钮
3. 输入新密码
4. 确认修改

## 远程连接

### 方式一：快速连接（免登录）

1. 点击首页「快速连接」
2. 输入对方设备的设备码
3. 输入设备密码
4. 点击「连接」

### 方式二：从设备列表连接

1. 登录账号
2. 进入「我的设备」页面
3. 点击要连接的设备的「连接」按钮
4. 如设备设置了密码，需要输入密码

### 连接成功后的操作

连接成功后，可进行以下操作：

| 操作 | 说明 |
|------|------|
| 远程桌面查看 | 查看对方桌面画面 |
| 屏幕控制 | 控制对方鼠标和键盘 |
| 文字聊天 | 与对方实时文字交流 |
| 断开连接 | 结束当前远程会话 |

## VIP服务

### VIP特权

| 功能 | 普通用户 | VIP用户 |
|------|---------|---------|
| 画质 | 标清 | 超清 |
| 连接速度 | 普通 | 专线加速 |
| 传输速度 | 限速 | 不限速 |
| 同时在线设备数 | 1台 | 3台 |
| 客服支持 | 无 | 7×24小时 |

### 开通VIP

1. 点击「VIP服务」或从菜单进入
2. 选择VIP套餐（月卡/季卡/年卡）
3. 点击「立即开通」
4. 选择支付方式并完成支付
5. VIP特权即时生效

### 查看VIP状态

登录后，在页面右上角用户头像旁边会显示VIP标识。

## 常见问题

### 连接失败

**可能原因：**
1. 设备码输入错误
2. 设备密码错误
3. 对方设备未在线
4. 网络连接不稳定

**解决方案：**
1. 确认设备码正确
2. 确认密码正确
3. 确认对方设备已启动并在线
4. 检查网络连接

### 画面卡顿

**解决方案：**
1. 尝试降低画质设置
2. 检查网络带宽
3. 开通VIP享受专线加速

### 无法绑定设备

**可能原因：**
1. 设备码不存在
2. 设备已被其他账号绑定
3. 设备已离线

**解决方案：**
1. 确认设备码正确
2. 先解绑原账号再重新绑定
3. 确保设备在线

### 忘记密码

通过绑定的邮箱找回密码，或联系客服。
